# YouTube Clone (HTML & CSS)

This project is a basic clone of the YouTube homepage built using HTML and CSS.

##  Project Overview
The goal of this project was to practice frontend development by recreating the layout and design of YouTube's interface.  
It focuses on layout structuring, styling, and responsive design principles.

##  Features
- Navigation bar
- Sidebar layout
- Video grid section
- Responsive design
- Clean UI structure

##  Technologies Used
- HTML
- CSS

##  What I Learned
- Flexbox & layout structuring
- Responsive design basics
- UI spacing & alignment
- Component-style layout thinking
